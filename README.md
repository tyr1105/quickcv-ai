## 🎯 QuickCV 能做什么？

**3步搞定简历优化：**

1. 📋 粘贴你的简历内容
2. 📝 粘贴目标职位描述（JD）
3. 🤖 AI自动分析匹配度 + 输出优化版简历

**核心功能：**

| 功能 | 说明 |
|------|------|
| 🔍 匹配度分析 | 0-100分评估，列出优势/劣势/缺失关键词 |
| ✨ 一键优化 | ATS关键词匹配 + 量化成就 + 强动词开头 |
| 🌐 中英双语 | 支持中文和英文简历输出 |
| 🔒 隐私安全 | 纯客户端运行，数据不上传服务器 |
| 💰 零成本 | 用户自带DeepSeek API Key，免费托管 |

## 🛠️ 技术栈

- **框架：** Next.js 16 + React 19 + TypeScript
- **样式：** Tailwind CSS 4（紫色渐变主题）
- **AI引擎：** DeepSeek API（纯客户端调用）
- **动画：** Framer Motion
- **部署：** GitHub Pages（GitHub Actions CI/CD）
- **成本：** $0（全部免费托管）

## 💡 使用场景

- 🎓 **应届生求职** — 第一次写简历，不知道怎么突出重点
- 🔄 **跳槽加薪** — 简历多年没更新，需要针对性优化
- 🏢 **ATS优化** — 简历总被系统过滤，需要匹配关键词
- 🌏 **海外求职** — 需要英文简历，中英双语一键切换
